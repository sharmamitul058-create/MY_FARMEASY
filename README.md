# Farmeasy 🌱

Farmeasy is a smart farming management system designed to help farmers and agricultural businesses manage crop planning, monitor farm resources, and track stock efficiently. The platform provides tools to organize farming activities and improve decision making.

---

## 🚀 Features

- Farmer Dashboard for managing farm activities
- Crop planning and prediction
- Farm stock management
- User authentication system
- Data storage and retrieval using backend server
- Simple and user-friendly interface
